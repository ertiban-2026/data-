# 🐝 MSc Thesis Data Repository

## Title

**Evaluation of Locally Formulated Supplemental Feeds for *Apis mellifera* During Dearth Periods in Waghimra, Ethiopia**

## Description

This repository contains raw and processed data used for my MSc research in Animal Production at the University of Gondar. The study evaluates the effectiveness of locally formulated supplemental feeds on honeybee colony performance during dearth periods.

## Contents

* `bee data corect.xlsx` – Primary dataset including:

  * Feed intake
  * Brood area
  * Colony strength
  * Honey yield
  * Nutritional composition

## Study Area

Waghimra Zone, Amhara Region, Ethiopia

## Methods Summary

* Experimental feeding trials
* Laboratory analysis (nutritional composition)
* Statistical analysis (ANOVA, model selection)
